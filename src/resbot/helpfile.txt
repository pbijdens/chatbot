/claim
If you're the first to do that you're the only admin of the bot in this channel

/admin &lt;add|delete|list&gt; &lt;userid&gt;
Add or remove chat admins

/whoami
Shows your user details

/whereami
Shows you where you are

/set &lt;command&gt; &lt;text&gt;
Sets a fixed reply when a user issues the command /command

/set nousername &lt;text&gt;
Sets a fixed reply when a user joins without having a username configured

/clear &lt;command&gt;
Removes the fixed reply for that command

/list
Lists the fixed commands

In addition the bot will send the text from the pinned message to every user that joins.
